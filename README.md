# Order Inventory System

A full-stack Order Inventory Management System to manage products, orders, and inventory efficiently.

## Tech Stack

Backend:
- Java
- Spring Boot
- Maven

Frontend:
- React

Database:
- MySQL / H2

## Features

- Create and manage orders  
- Inventory tracking  
- Product management  
- REST API integration  
- Full-stack implementation (Frontend + Backend)

## Project Structure

```
order-inventory/
│
├── cg-Order-Inventory/            # Backend (Spring Boot)
├── cg-Order-Inventory-Frontend/   # Frontend (React)
├── README.md
```

## Setup Instructions

### Backend Setup

1. Navigate to backend folder:
```
cd cg-Order-Inventory
```

2. Run the application:
```
mvn spring-boot:run
```

Backend runs on:
```
http://localhost:8080
```

---

### Frontend Setup

1. Navigate to frontend folder:
```
cd cg-Order-Inventory-Frontend
```

2. Install dependencies:
```
npm install
```

3. Start frontend:
```
npm start
```

Frontend runs on:
```
http://localhost:3000
```

---
